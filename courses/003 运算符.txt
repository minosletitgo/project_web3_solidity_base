
算术运算符：
    + （加）
    +=（加等于）
    - （减）
    -=（减等于）
    * （乘）
    / （除）
    % （取模）

比较运算符：
    == （相等）
    != （不相等）
    > （大于）
    < （小于）
    >= （大于等于）
    <= （小于等于）

逻辑运算符：
    && （与）
    || （或）
    ! （非）    

三元运算符：
    // return the max of x and y
    return x >= y ? x: y; 

幂运算符：
    // 2的3次方
    uint result = 2 ** 3;     
    // 10的18次方(Solidity版本号^0.8.0)
    uint result = 10 ** 18;
