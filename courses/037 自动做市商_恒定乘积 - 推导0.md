
#### 详细推导```function swap```的数学计算
#### 基础数据介绍
- $x$，代表当前流动池现有的代币A
- $y$，代表当前流动池现有的代币B
- $△x$，代表用户放入进的代币A
- $△y$，代表用户拿出的代币B

　

#### 使用"默认恒定"公式
- $x·y = (x + △x)·(y + △y)$
- $△y = (△x · y) / (x + △x)$
- $△y = (△x · y)·(100$\% - 0.3\%) / (x + △x)
- $△y = (△x · y)·(99.7$\%) / (x + △x)
- $△y = (△x · y)·997 / (x + △x)·1000$
